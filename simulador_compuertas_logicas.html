S = A · B

AND produce 1 únicamente cuando ambas entradas valen 1.

Circuito combinado de dos niveles

Construye una expresión del tipo S = G2(G1(A,B), C). Esto permite observar cómo se encadenan compuertas para formar circuitos lógicos más complejos.

X = A · B | S = X · C

El primer nivel calcula X. Luego X se combina con C en una segunda compuerta para obtener la salida final S.

Guía conceptual

Compuerta lógica: dispositivo o abstracción computacional que recibe una o más entradas binarias y produce una salida binaria según una regla booleana.

Tabla de verdad: representación exhaustiva de todas las combinaciones posibles de entrada y su salida correspondiente.

Expresión booleana: forma simbólica de representar la operación lógica. Por ejemplo, AND puede escribirse como S = A · B.

Actividad sugerida para estudiantes

  1. Selecciona una compuerta y completa manualmente su tabla de verdad antes de usar el simulador.
  2. Activa las entradas A y B para comprobar si tu predicción fue correcta.
  3. Explica con tus palabras en qué casos la salida es 1 y en qué casos es 0.
  4. En el modo circuito combinado, construye una expresión de dos niveles y describe el proceso lógico paso a paso.
  5. Propón un caso cotidiano que pueda representarse con una compuerta lógica, por ejemplo: acceso permitido, alarma, votación o encendido automático.

Preguntas de reflexión

  1. ¿Qué diferencia conceptual existe entre OR y XOR?
  2. ¿Por qué NAND y NOR se consideran compuertas relevantes para construir circuitos?
  3. ¿Qué ocurre cuando se encadenan dos compuertas?
  4. ¿Cómo se relaciona una tabla de verdad con la toma de decisiones computacional?
  5. ¿Qué limitaciones tiene este simulador frente a un circuito físico real?
